# Wind Turbine Power Prediction using LSTM

This project builds an industry-focused LSTM time-series model to predict next-step (t+1) wind turbine power output using SCADA data.

## Objectives
- Predict turbine power output using historical SCADA measurements
- Apply LSTM-based time-series modeling
- Deploy predictions via a Streamlit dashboard

## Tech Stack
- Python
- TensorFlow / Keras
- Pandas, NumPy
- Scikit-learn
- Streamlit
